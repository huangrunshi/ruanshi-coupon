# 润石券宝 - 腾讯龙虾自动化开发需求清单

## 一、优先落地（核心必做，解决人工依赖）

### 1. 订单支付成功后自动执行权益发放
- 对接视频会员、音乐会员、话费等供应商API，实现自动直充到账（手机号/账号），无需人工操作
- 卡券类权益自动生成/拉取卡密，加密后自动推送至用户中心/短信/微信模板消息
- 充值/发卡结果自动回写订单状态（成功/失败/处理中）
- 失败自动触发重试/补发流程
- 支持批量订单并发处理，保证7×24小时稳定运行

### 2. 订单全链路状态自动同步与消息推送
- 支付成功→自动生成订单号→自动触发发货流程
- 发货中/已到账/失败→自动更新订单状态
- 每一步状态变更，自动通过微信ClawBot/公众号模板消息/短信推送用户
- 支持用户在微信/网页端查询订单

### 3. 权益库存、价格、上下架状态自动同步
- 定时/实时拉取供应商库存、官方价、活动价，自动更新平台展示价
- 库存不足/售罄时，自动下架对应权益，禁止下单
- 价格变动时，自动刷新前端展示，确保"全网低价"实时有效
- 异常库存/价格波动自动告警（微信/邮件）

### 4. 24小时自动客服
- 导入产品手册、售后规则、常见问题（FAQ）作为知识库
- 实现意图识别，自动识别用户问题（充值失败、怎么用、有效期、售后、价格）
- 自动生成标准化回答，支持图文/链接/按钮交互
- 复杂问题自动转接人工，并保留上下文

## 二、次要落地（降本提效，完善闭环）

### 1. 售后工单自动处理
- 用户提交"充值失败/卡密无效"→自动核验订单与权益状态
- 符合条件自动执行补发卡密/重新充值/原路退款
- 处理结果自动通知用户，并生成售后记录
- 高频售后自动统计，用于优化供应链与产品

### 2. 券宝专属智能顾问能力
- 自动推荐热门/低价权益（基于用户偏好/销量）
- 自动比价，展示平台价vs官方价，突出优惠
- 自动解答会员时长、使用限制、有效期、售后规则
- 用户发"我的订单"，自动返回最近5条订单状态

### 3. 风控规则自动执行
- 同一手机号/IP/设备，每日/每周限购自动限制
- 新用户首次下单自动触发短信/微信验证
- 异常订单（批量下单、异地、高频）自动标记并拦截，人工复核
- 卡密自动加密展示，防止泄露盗刷

## 三、后续落地（运营增长，长期优化）

### 1. 营销活动自动化
- 新人专享价、限时秒杀、组合套餐，自动生效/自动下架
- 满减/折扣规则自动计算，下单时自动应用
- 活动开始/结束自动推送消息（微信/短信）

### 2. 运营数据看板
- 自动统计曝光→点击→下单→支付→到账→售后全链路转化率
- 自动生成每日/每周销量、热门权益、用户画像、活动ROI
- 数据自动同步至后台/Excel/企业微信，支持定时推送

### 3. 用户中心自动化
- 我的订单、我的卡密、我的积分、售后记录自动同步与展示
- 卡密过期自动提醒，积分自动累计与抵扣
- 一键兑换/充值入口自动生成

## 四、技术部署与集成要求

1. 优先使用腾讯龙虾ADP/WorkBuddy/QClaw能力，基于OpenClaw框架开发
2. 所有自动化流程用可视化工作流（Workflow）配置，便于维护
3. 对接微信ClawBot/公众号/企业微信，实现微信内一站式服务
4. 所有接口与流程做异常捕获、重试、日志、告警，保证稳定

## 五、整体核心要求

把「润石券宝」从"人工为主"升级为"全自动闭环"：
- 支付后自动发货/充值，去掉人工联系发货
- 24小时自动客服，减少人工咨询
- 库存/价格/订单自动同步，防超卖、保体验
- 风控/售后自动处理，降成本、提信任
- 所有流程可监控、可告警、可快速迭代
