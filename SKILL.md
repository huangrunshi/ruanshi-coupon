# 润石券宝 - 权益智能顾问

## 概述
润石券宝是一个权益优惠券智能推荐系统，帮助用户快速找到合适的视频会员、外卖券、音乐会员等数字权益产品。

## 功能特性
- 分类浏览：视频会员、外卖餐饮、音乐会员、生活服务、办公效率、游戏娱乐
- 关键词搜索：支持按名称、品牌、描述搜索
- 热门推荐：自动展示热卖产品
- 价格对比：显示原价、现价及折扣力度

## 数据更新
产品数据存储在 `index.html` 的 `PRODUCTS` 数组中。
如需更新产品，请编辑该数组，格式如下：
```javascript
{id:"唯一ID",name:"产品名称",price:现价,original:原价,category:"分类",sub:"子分类",period:"时长",desc:"描述",url:"购买链接",hot:是否热门}
```

## 分类列表
- 视频会员：爱奇艺、腾讯视频、优酷、B站
- 外卖餐饮：美团、饿了么、瑞幸、肯德基、麦当劳
- 音乐会员：网易云、QQ音乐、酷狗
- 生活服务：滴滴出行等
- 办公效率：WPS、有道云笔记
- 游戏娱乐：各类游戏点卡
